# React with Webpack
React starter kit running on Webpack.
Implemented:
- Sass
- File loader
- URL loader
- Redux

## How to install
`git clone https://github.com/seeseesky/react_webpack.git`

`npm install`

## How to run
`npm start`

## How to build
`npm run build`
